#include <stdexcept>
udostępnia standardowe klasy wyjątków, które są używane do sygnalizowania błędów w czasie działania programu.

W ćwiczeniu 2 jest on potrzebny do obsługi wyjątku:
	std::out_of_range

rzucanego przez metodę:
	deque::at()

2. Kiedy deque::at() zgłasza wyjątek
	d.at(i)

rzuca wyjątek std::out_of_range, gdy:
	i < 0 (pośrednio, np. po konwersji typu),
	i >= d.size().
Przykład sytuacji błędnej:

std::deque<int> d(10);
	d.at(10);   // błąd – ostatni poprawny indeks to 9

3. Poprawna obsługa wyjątku (schemat)
	try {
		// kod, który może zgłosić wyjątek
	}
	catch (const std::out_of_range& e) {
		// reakcja na błąd
	}
