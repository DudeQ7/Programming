
# 🗂️ Programming Archive Repository
This repository serves as a **personal code archive** and **experimentation space** for various programming languages, technologies, and concepts.
Instead of maintaining many separate repositories, each language or topic is stored in its **own dedicated branch**.
---
## 📌 Repository Concept
- Each **branch = one programming language / technology / topic**
-This repo acts as a **collection of examples** for personal purposes
---
## 🌿 Branch Structure
Example convention:
```text
branch-name/
├── README.md        # description of the branch / language / goal
├── examples/        # code examples
├── notes/           # notes, comments, TODOs
└── experiments/     # quick experiments and prototypes
```
